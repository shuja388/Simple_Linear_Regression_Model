# Salary Prediction using Linear Regression

This project predicts salary based on years of experience using Machine Learning Linear Regression.

## Features
- Data preprocessing
- Train/Test split
- Linear Regression model
- Data visualization
- Prediction on unseen data
- Model evaluation

## Installation

Clone repository:

```bash
git clone https://github.com/yourusername/salary-prediction-linear-regression.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run project:

```bash
python salary_prediction.py
```

## Dataset

Place `Salary_Data.csv` inside the `data/` folder.

Dataset columns:

- YearsExperience
- Salary
